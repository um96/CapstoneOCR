OCR Conversion

Convert Image files into Text/Word files

What's the Problem?

You are someone looking for a way to simplify time and work needed to get text from a file.

Imagine as a student, you missed a class and you ask your friend for their notes. They send it over to you, a screenshot of their notes. You want to add some extra notes to it but thats when you realize: you can't type in notes into that screenshot. Now, you are constantly thinking if you should type all the notes your friend gave you into a Word Document and include your own notes, or print it out and write notes on the side. This is definitely a lot of work, especially when you don't have much time to type it up.

Businesses and many offices, also, may need to get text specifically from documents that are faxed or scanned into image files. It is inonvenient if they have to retype every single character into a new document. It will be very convenient if they can just get all the text from the file and be able to edit it from there, without retyping all the information.

This is why we have OCR.

What is OCR?

OCR stands for Optical Character Recognition.

It is a method of obtaining text from an image file or a file where copying the text is difficult. OCR recognizes the text in an image file, takes it and returns you the actual text. The text is not as a screenshot of where the text is, but the actual machine-encoded text, which you can copy and paste.
